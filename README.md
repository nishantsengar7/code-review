# code-review

🤖 AI-Powered Code Review Bot

An AI-driven code review assistant built with Node.js and React.js, powered by Google Gemini API.
It automatically reviews your code, detects potential issues, suggests improvements, and enhances code quality in real time.
✨ Features
🔍 Automated Code Analysis – Detects syntax errors, bad practices, and inefficiencies.
💡 Smart Suggestions – Improves readability, performance, and security.
📊 Detailed Review Reports – Explains issues and provides example fixes.
🌐 Web-Based Interface – User-friendly frontend built with React.js.
⚡ Real-Time Feedback – Powered by Google Gemini AI for instant results.
🛠️ Tech Stack
Frontend: React.js, Prism.js (for syntax highlighting), Tailwind CSS (optional)
Backend: Node.js, Express.js
AI API: Google Gemini API
